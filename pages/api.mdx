# API Reference

Complete API documentation for Nexus Trading platform.

## Authentication

```javascript
const nexus = new NexusTrading({
  apiKey: 'your-api-key',
  environment: 'production' // or 'sandbox'
});
```

## Core Endpoints

### Historical Data

```javascript
GET /api/v1/historical
```

Parameters:
- `symbol` (required): Stock symbol
- `interval`: Time interval (1m, 5m, 1h, 1d)
- `period`: Historical period (1d, 1mo, 1y)
- `adjusted`: Boolean for adjusted prices

### Technical Analysis

```javascript
GET /api/v1/analysis
```

Parameters:
- `symbol` (required): Stock symbol
- `indicators`: Array of technical indicators
- `aiEnhanced`: Boolean for AI enhancement
- `period`: Analysis period

### ML Predictions

```javascript
GET /api/v1/predict
```

Parameters:
- `symbol` (required): Stock symbol
- `model`: Prediction model type
- `horizon`: Prediction timeframe
- `confidence`: Include confidence scores

## Rate Limits

- Free tier: 100 requests/minute
- Pro tier: 1000 requests/minute
- Enterprise tier: Custom limits

## Error Codes

- `401`: Authentication error
- `403`: Permission denied
- `429`: Rate limit exceeded
- `500`: Internal server error

## WebSocket API

```javascript
// Connect to real-time feed
const ws = nexus.connectWebSocket({
  symbols: ['AAPL', 'GOOGL'],
  channels: ['trades', 'quotes']
});

ws.on('message', (data) => {
  console.log('Received:', data);
});
```
